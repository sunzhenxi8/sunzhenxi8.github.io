Create a new video post with:

Then edit the generated file in source/_posts/. The key field is embed, which should be a direct embeddable player URL such as a YouTube /embed/ link, Vimeo player URL, Internet Archive embed URL, or another source that allows embedding.

Use cover when you want a custom poster image. If cover is empty and the link is from YouTube, the theme will try to use the YouTube thumbnail automatically.
